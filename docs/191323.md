• I. $ \{ x|{x <= a} \cup {b <= x <= c}, a<b<c \in \mathbb{R} \} $
• II. $ \{ x|{a <= x <= b} \cup {x >= c}, a<b<c \in \mathbb{R} \} $


A. 4

D. Infinite
E. None of the above
